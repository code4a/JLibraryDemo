# JLibrary

抽取的Activity和Fragment基类以及一些工具类!

* 新增bugly的bug上报和应用更新！通过JbuglyManger修改appid和appkey 并初始化bugly

# 如何使用

#### Gradle
```
dependencies {
  compile 'com.code4a:jlibrary:1.0.3'
}
```

#### Maven
```
<dependency>
  <groupId>com.code4a</groupId>
  <artifactId>jlibrary</artifactId>
  <version>1.0.3</version>
  <type>pom</type>
</dependency>
```
